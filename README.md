# CodeAlpha URL Shortener

This is a simple URL Shortener backend project built using Node.js and Express.

## Features
- Shortens long URLs
- Redirects short URL to original URL
- Built using Express.js

## How to Run
1. Install Node.js
2. Run `npm install`
3. Run `node index.js`
4. Open browser: http://localhost:3000
